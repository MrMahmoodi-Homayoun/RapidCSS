Installation

Method 1: Download

Download the CSS file and include it in your project:

<link rel="stylesheet" href="rapid-v1.0.css">

Utility Classes

RapidCSS includes a comprehensive set of utility classes for spacing, text alignment, display properties, and more.

Spacing Utilities

Use margin and padding utilities to control spacing:

<div class="mt-3">Margin top</div> <div class="mb-3">Margin bottom</div> <div class="ml-3">Margin left</div> <div class="mr-3">Margin right</div> <div class="p-3">Padding all sides</div>

Text Utilities

Control text alignment, weight, and color:

<p class="text-left">Left aligned text</p> <p class="text-center">Center aligned text</p> <p class="text-right">Right aligned text</p> <p class="font-weight-bold">Bold text</p> <p class="text-primary">Primary colored text</p>

Customization

RapidCSS is designed to be easily customizable to match your brand identity.

CSS Variables

Override the default CSS variables to customize the appearance:

:root { --primary: #your-color; --secondary: #your-color; --success: #your-color; --danger: #your-color; --warning: #your-color; --info: #your-color; --light: #your-color; --dark: #your-color; --white: #your-color; --border: #your-color; --radius: 0.5rem; --shadow: 0 0.125rem 0.25rem rgba(0, 0, 0, 0.1); }

Creating Custom Components

Extend RapidCSS with your own custom components:

/* Custom alert component */ .alert { padding: 1rem; border-radius: var(--radius); margin-bottom: 1rem; border: 1px solid transparent; } .alert-success { background-color: #d4edda; color: #155724; border-color: #c3e6cb; } .alert-danger { background-color: #f8d7da; color: #721c24; border-color: #f5c6cb; }

SASS Customization

If you're using SASS, you can customize the framework by modifying the source variables:

// Override SASS variables $primary: #your-color; $secondary: #your-color; $border-radius: 0.5rem; // Import RapidCSS @import "rapid";
